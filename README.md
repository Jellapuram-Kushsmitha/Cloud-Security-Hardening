## Overview
A comprehensive 14-phase cloud security hardening project
covering Linux hardening, Docker security, AWS IAM, Terraform
IaC, and automated DevSecOps pipelines.

## Security Controls
- SSH hardening with key-only authentication
- UFW firewall with default deny policy
- Fail2Ban intrusion prevention
- Lynis security audit score: 79/100
- Zero rootkits detected (rkhunter + chkrootkit)
- Container images scanned with Trivy — CVE-2026-42496 fixed
- Terraform IaC with Checkov security scanning
- Automated pipeline: secret detection + CVE scan + IaC scan
